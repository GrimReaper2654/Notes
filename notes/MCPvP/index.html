    # Minecraft Commands by Choyu

    I sometimes mess around with minecraft commands so I thought I would document some of the stuff I make. Previously, I had lost several hundred of these commands when my computer died and I lost all the files on my computer so this time, I’m uploading them to github. Anyways, these commands are designed for the latest snapshot of minecraft (which has all the cool armour trims and stuff) but they shoud work in normal 1.19 or lower versions. These commands are all quite long so they will need to be run in a command block (which you get by typing /give @s command_block 1 in the chat with cheats enabled). All the items on this page are quite OP compared to vanilla items but actually surprisingly ballanced with eachother. Some npc mobs have been added at the end of the page in case you have no friends to play with (like me). Hopefully, they provide sufficient entertainment for you.
